## Devlog
1.) Nodes could be like ingredients while Scenes are a saved recipe for a specific dish. In my analogy Nodes are the basic unit with only one specific job. By themsleves there simple. But when combing these Nodes you make Scenes just like how you combine different ingredients to make a cake or any other food. With the "recipe" or the Scene made, it makes reusing it much more easier.
